## Get started with Rust

Download rustup: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

`cargo init`

And...you are done.

## To create docs

`cargo doc`

## To run unit tests

`cargo test`

## Features of rust

* No runtime
* No garbage collector
* No dangling pointers (and many other C++/C issues)
